========4. 머신러닝========
머신러닝 : '학습'을 컴퓨터로 구현한 것. -> 규칙 찾기
  교사 / 비교사 / 강화 : 답o / 답x / 답△

주요 framework
  scikit-learn : 머신러닝 그 자체. svm 많이 씀. 랜덤포레스트도 씀.
  pandas : 데이터분류할때 씀.
  matplotlib : 데이터 시각화

예시
  이미지판별
  언어판별

데이터 검증
  3-fold cross validation 많이 씀.
  grid search로 매개변수 튜닝해 효율 상승 ㄱㄴ.

========5. 딥러닝========
딥러닝 : '특징량을 알아서 추출하는' 머신러닝.
  기존에는 이미지분석시 색에 주목하라는 특징을 지정했다면
  딥러닝은 알아서 input을 기반으로 색에 주목함.
  DNN(깊은 신경망)을 사용하기 때문에 딥러닝임.

딥러닝을 위한 개념
  신경망 : 입력층 - 중간층 - 출력층으로 이뤄진, 여러 뉴런이 연결된 구조
  퍼셉트론 : 인공 신경망
    가중치와 역치를 반영해 의사결정

tensorflow : 대규모 숫자 계산을 해주는 라이브러리
  간단한 과정 : 상수 정의 -> 계산식 정의 -> 세션 시작
  조금더? : 상수&변수정의 -> 데이터플로우그래프 정의 -> 세션실행
  계산을 '그래프'로 처리함.
  플레이스홀더는.. 그냥 버퍼같은 건가보다.
  텐서플로 사용법은 scikit-learn보다 복잡함. 그래서 나온 것이 시각화된 tensorboard

tensorflow로 딥러닝

  tensorflow보다 쉽게 딥러닝 ㄱㄴ.

  numpy : 행렬 연산 간단히 ㄱㄴ.
